# TARAZU

A hyperlocal price and inventory comparison React Native application for Indian quick commerce and e-commerce platforms. Compare prices, delivery availability, and stock across multiple services in real time.

## Features
* **Hyperlocal Comparison:** Compare platforms like Blinkit, Zepto, Swiggy Instamart, BigBasket, JioMart, Amazon, Flipkart, and more.
* **Category Filtering:** Filter platforms based on categories (Quick Delivery, Standard, Food, Electronics, Medicine, etc.).

## Download

## Development Setup
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the Expo development server:
   ```bash
   npm run start
   ```
